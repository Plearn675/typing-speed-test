# Typing Speed Test 🧠💨

This is a simple Python desktop app built with **Tkinter** that tests your typing speed.  
It measures how many words you can type in 60 seconds.

---

## 🚀 Features
- Start and stop timer (60 seconds)
- Word counter
- Reset function
- Clean and responsive Tkinter UI

---

## 🧰 Technologies
- Python 3
- Tkinter (GUI)
